The idea behind this method is simplify the creation of menu options.

The name of the menu item should be Parent.Item to refer to parent, and also the prefix cb_ and rb_ shoulb be used to create JCheckBoxMenuItem and JRadioButtonMenuItem respectively Example: addItem("File", "File"); addItem("File.SaveAs", "Save as...", "save_as.gif")

Returns
the created MenuItem, null if an error occurs

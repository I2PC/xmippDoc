43 class XmippProtML2D(ProtClassify2D):
44  """
45  Perform (multi-reference) 2D-alignment using
46  a maximum-likelihood ( *ML* ) target function.
47 
48  Initial references can be generated from random subsets of the experimental
49  images or can be provided by the user (this can introduce bias). The output
50  of the protocol consists of the refined 2D classes (weighted averages over
51  all experimental images). The experimental images are not altered at all.
52 
53  Although the calculations can be rather time-consuming (especially for
54  many, large experimental images and a large number of references we
55  strongly recommend to let the calculations converge.
56  """
57  _label = 'ml2d'

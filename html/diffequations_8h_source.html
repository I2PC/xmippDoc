132 /*************************************************************************
133 Cash-Karp adaptive ODE solver.
134 
135 This subroutine solves ODE Y'=f(Y,x) with initial conditions Y(xs)=Ys
136 (here Y may be single variable or vector of N variables).
137 
138 INPUT PARAMETERS:
139  Y - initial conditions, array[0..N-1].
140  contains values of Y[] at X[0]
141  N - system size
142  X - points at which Y should be tabulated, array[0..M-1]
143  integrations starts at X[0], ends at X[M-1], intermediate
144  values at X[i] are returned too.
145  SHOULD BE ORDERED BY ASCENDING OR BY DESCENDING!!!!
146  M - number of intermediate points + first point + last point:
147  * M>2 means that you need both Y(X[M-1]) and M-2 values at
148  intermediate points
149  * M=2 means that you want just to integrate from X[0] to
150  X[1] and don't interested in intermediate values.
151  * M=1 means that you don't want to integrate :)
152  it is degenerate case, but it will be handled correctly.
153  * M<1 means error
154  Eps - tolerance (absolute/relative error on each step will be
155  less than Eps). When passing:
156  * Eps>0, it means desired ABSOLUTE error
157  * Eps<0, it means desired RELATIVE error. Relative errors
158  are calculated with respect to maximum values of Y seen
159  so far. Be careful to use this criterion when starting
160  from Y[] that are close to zero.
161  H - initial step lenth, it will be adjusted automatically
162  after the first step. If H=0, step will be selected
163  automatically (usually it will be equal to 0.001 of
164  min(x[i]-x[j])).
165 
166 OUTPUT PARAMETERS
167  State - structure which stores algorithm state between subsequent
168  calls of OdeSolverIteration. Used for reverse communication.
169  This structure should be passed to the OdeSolverIteration
170  subroutine.
171 
172 SEE ALSO
173  AutoGKSmoothW, AutoGKSingular, AutoGKIteration, AutoGKResults.
174 
175 
176  -- ALGLIB --
177  Copyright 01.09.2009 by Bochkanov Sergey
178 *************************************************************************/

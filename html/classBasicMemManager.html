Obtain raw memory block. This might lead to memory allocation. Does not ensure that the memory is actually allocated (i.e. you might receive nullptr). Consider zero-ing it before use. This method does not offer it on purpose as e.g. cuda memory can be memset asynchronously in specific stream.

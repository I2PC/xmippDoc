This function will try to find where the images are located relative to the container metadata, the current dir or the project dir.

The search will be in the following order: 1. From current dir (default behaivour if this function is not used) 2. Relative to metadata 3. Relative to the project root folder

Parameters
filenameimage filename (probably from a metadata)
mdFilenamemetadata filename, used to check if there are relative to there
shouldExist
Returns
The path of the image if found, or null if not exists

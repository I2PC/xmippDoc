430 /*************************************************************************
431 Dense solver.
432 
433 This subroutine solves a system A*x=b, where A is NxN non-denegerate
434 real matrix, x and b are vectors.
435 
436 Algorithm features:
437 * automatic detection of degenerate cases
438 * condition number estimation
439 * iterative refinement
440 * O(N^3) complexity
441 
442 INPUT PARAMETERS
443  A - array[0..N-1,0..N-1], system matrix
444  N - size of A
445  B - array[0..N-1], right part
446 
447 OUTPUT PARAMETERS
448  Info - return code:
449  * -3 A is singular, or VERY close to singular.
450  X is filled by zeros in such cases.
451  * -1 N<=0 was passed
452  * 1 task is solved (but matrix A may be ill-conditioned,
453  check R1/RInf parameters for condition numbers).
454  Rep - solver report, see below for more info
455  X - array[0..N-1], it contains:
456  * solution of A*x=b if A is non-singular (well-conditioned
457  or ill-conditioned, but not very close to singular)
458  * zeros, if A is singular or VERY close to singular
459  (in this case Info=-3).
460 
461 SOLVER REPORT
462 
463 Subroutine sets following fields of the Rep structure:
464 * R1 reciprocal of condition number: 1/cond(A), 1-norm.
465 * RInf reciprocal of condition number: 1/cond(A), inf-norm.
466

468  Copyright 27.01.2010 by Bochkanov Sergey

1481 /*************************************************************************
1482  LEVENBERG-MARQUARDT-LIKE NONLINEAR SOLVER
1483 
1484 DESCRIPTION:
1485 This algorithm solves system of nonlinear equations
1486  F[0](x[0], ..., x[n-1]) = 0
1487  F[1](x[0], ..., x[n-1]) = 0
1488  ...
1489  F[M-1](x[0], ..., x[n-1]) = 0
1490 with M/N do not necessarily coincide. Algorithm converges quadratically
1491 under following conditions:
1492  * the solution set XS is nonempty
1493  * for some xs in XS there exist such neighbourhood N(xs) that:
1494  * vector function F(x) and its Jacobian J(x) are continuously
1495  differentiable on N
1496  * ||F(x)|| provides local error bound on N, i.e. there exists such
1497  c1, that ||F(x)||>c1*distance(x,XS)
1498 Note that these conditions are much more weaker than usual non-singularity
1499 conditions. For example, algorithm will converge for any affine function
1500 F (whether its Jacobian singular or not).
1501 
1502 
1503 REQUIREMENTS:
1504 Algorithm will request following information during its operation:
1505 * function vector F[] and Jacobian matrix at given point X
1506 * value of merit function f(x)=F[0]^2(x)+...+F[M-1]^2(x) at given point X
1507 
1508 
1509 USAGE:
1510 1. User initializes algorithm state with NLEQCreateLM() call
1511 2. User tunes solver parameters with NLEQSetCond(), NLEQSetStpMax() and
1512  other functions
1513 3. User calls NLEQSolve() function which takes algorithm state and
1514  pointers (delegates, etc.) to callback functions which calculate merit
1515  function value and Jacobian.
1516 4. User calls NLEQResults() to get solution
1517 5. Optionally, user may call NLEQRestartFrom() to solve another problem
1518  with same parameters (N/M) but another starting point and/or another
1519  function vector. NLEQRestartFrom() allows to reuse already initialized
1520  structure.
1521 
1522 
1523 INPUT PARAMETERS:
1524  N - space dimension, N>1:
1525  * if provided, only leading N elements of X are used
1526  * if not provided, determined automatically from size of X
1527  M - system size
1528  X - starting point
1529 
1530 
1531 OUTPUT PARAMETERS:
1532  State - structure which stores algorithm state
1533 
1534 
1535 NOTES:
1536 1. you may tune stopping conditions with NLEQSetCond() function
1537 2. if target function contains exp() or other fast growing functions, and
1538  optimization algorithm makes too large steps which leads to overflow,
1539  use NLEQSetStpMax() function to bound algorithm's steps.
1540 3. this algorithm is a slightly modified implementation of the method
1541  described in 'Levenberg-Marquardt method for constrained nonlinear
1542  equations with strong local convergence properties' by Christian Kanzow
1543  Nobuo Yamashita and Masao Fukushima and further developed in 'On the
1544  convergence of a New Levenberg-Marquardt Method' by Jin-yan Fan and
1545  Ya-Xiang Yuan.
1546

1549  Copyright 20.08.2009 by Bochkanov Sergey
1550 *************************************************************************/

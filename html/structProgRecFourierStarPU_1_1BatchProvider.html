Detailed Description

Implements a potentially lazy collection of batches for computeStarPU. Standard, single machine operation will implement this with a simple counter, distributed version may retrieve the batches over the network, etc.

Also tracks the progress, via batchCompleted(), possibly showing and updating a progress bar. While each batch may take a different amount of time, it is a good and fast approximation.

56 class XmippProtReconstructSwarm(ProtRefine3D):
57  """This is a 3D refinement protocol whose main input is a set of volumes and a set of particles.
58  The set of particles has to be at full size (the finer sampling rate available), but
59  the rest of inputs (reference volume and masks) can be at any downsampling factor.
60  The protocol scales the input images and volumes to a size that depends on the target resolution.
61 
62  The input set of volumes is considered to be a swarm of volumes and they try to optimize
63  the correlation between the volumes and the set of particles. This is an stochastic maximization
64  and only a fraction of the particles are used to update the volumes and evaluate them.
65  """
66  _label = 'swarm consensus'
67  _version = VERSION_2_0

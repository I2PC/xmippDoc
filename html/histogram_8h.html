Functions related to histograms 2D

The vectors can be of any numerical type (short int, int, double, ...), but both of the same type.

Vectors must be of the same shape, the first element of v1 and the first of v2 define the position were the first point will be inserted in the histogram, then the second of v1 and of v2, ... That is, the elements of v1 and v2 serve as coordinates within the histogram. The number of steps must always be given.

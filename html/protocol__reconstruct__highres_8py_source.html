64 class XmippProtReconstructHighRes(ProtRefine3D, HelicalFinder):
65  """This is a 3D refinement protocol whose main input is a volume and a set of particles.
66  The set of particles has to be at full size (the finer sampling rate available), but
67  the rest of inputs (reference volume and masks) can be at any downsampling factor.
68  The protocol scales the input images and volumes to a reasonable size depending on
69  the resolution of the previous iteration.
70 
71  The protocol works with any input volume, whichever its resolution, as long as it
72  is a reasonable initial volume for the set of particles. The protocol does not
73  resolve the heterogeneous problem (it assumes an homogeneous population),
74  although it is somewhat tolerant through the use of particle weights in the
75  reconstruction process.
76 
77  It is recommended to perform several global alignment iterations before entering
78  into the local iterations. The switch from global to local should be performed when
79  a substantial percentage of the particles do not move from one iteration to the next.
80 
81  The algorithm reports the cross correlation (global alignment) or cost (local) function
82  per defocus group, so that we can see which was the percentile of each particle in its
83  defocus group. You may want to perform iterations one by one, and remove from one
84  iteration to the next, those particles that worse fit the model."""
